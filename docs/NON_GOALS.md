# Non-Goals

Do not implement any of the following in the first pass:

- App UI
- SwiftUI app shell
- Continuity Studio
- CivicTrace
- SourceTrace
- yLLMKit integration
- LLM provider selection
- Prompt templates
- PDF parsing
- Ebook parsing
- OCR
- Audio transcription
- Video transcription
- Image metadata extraction
- Web crawling
- Source file copying and checksum verification
- Embeddings
- Semantic search
- SQLite FTS5
- Custom graph query language
- Custom fictional calendars
- Domain registries
- Conflict checker
- Import batch contract
- Export/import package support
- Audit history
- Cloud sync
- Multi-user collaboration
- Permissions
- Encryption
- Digital signatures
- Undo or rollback
